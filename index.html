RECENT PROJECTS

Golf Handicap Calculator +

This app keeps a history of a user's rounds of golf & automatically calculates and updates their handicap index as they add new rounds. It uses JavaScript, jQuery, Sass on the front end and the API was built with Ruby on Rails for a PostgreSQL database.

Stranger Tic Tac Toe

This is the first app I ever built and is a simple tic tac toe game with a great Stranger Things theme. It's logic is written in JavaScript and is styled and animated with jQuery and custom Sass. Nelson Cash designed the title font and background.

Sweaters for Turtles

A team built, full stack E-commerce web app for funny sweaters knit for turtles. The front end is built with JavaScript, jQuery Handlebars, BootStrap and Sass. The back end was built using Node.js, Express.js, Mongoose and uses MongoDB.

Friendsofthaidaughters.org

A website for a non-profit organization that fights to prevent sex trafficking in the Golden Triangle. Originally developed with custom HTML and CSS, it has since been redesigned on Wix to allow editing by others in the organization.
